
# Shell Asia Terminal Based

Merupakan sebuah program yang terinspirasi dari aplikasi Shell Asia , yang dimana kita bisa menukarkan Point Shell dengan Item yang ada di Redeem Shop


## Authors

- [@justshev](https://www.github.com/justshev)


## Features

- Membuat akun sebagai Admin & User
- Admin bisa menambahkan data user, menghapus data user, melihat data seluruh user, menambahkan item kedalam redeem shop, memperbarui data item yang ada di redeem shop, menambahkan point kedalam akun user
- User bisa melihat data seluruh item yang ada didalam redeem shop, menukarkan point dengan item yang ada didalam redeem shop, mencari item dengan berdasarkan kategori, mencari item dengan berdasarkan nama, mengganti password user
